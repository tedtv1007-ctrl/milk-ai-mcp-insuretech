# Milk AI MCP InsureTech Server

This project implements a Model Context Protocol (MCP) Server specifically designed for the Insurance industry (InsureTech), focusing on secure claim processing and policy analysis.

## Core Features (Roadmap)
1. **PII Sanitization Layer**: Ensures sensitive data is masked before being sent to public LLMs.
2. **Policy Verification Tools**: Automated checks against standard insurance terms.
3. **Claim Validation**: OCR integration and preliminary risk assessment.

## Security First
Designed for financial environments with strict DLP (Data Loss Prevention) requirements.
